# NGESS School Website - Design System

## 🎨 Color Palette (Extracted from NGESS Logo)

### Primary Brand Colors
- **Primary Green**: `#00864B` - Professional, trustworthy forest green
- **Primary Dark**: `#005C35` - Darker shade for hover/active states
- **Primary Light**: `#00A85F` - Lighter accent shade

### Secondary Colors (From Logo)
- **Accent Medium**: `#66B693` - Medium green for secondary elements
- **Accent Soft**: `#B4DBCA` - Soft green for backgrounds
- **Background Light**: `#E1F1EA` - Almost-white green for subtle backgrounds

### Neutral Palette
- **White**: `#FFFFFF`
- **Off-White**: `#F8FAFB`
- **Light Gray**: `#F1F5F7`
- **Medium Gray**: `#B8C5D0`
- **Dark Gray**: `#4A5568`
- **Near-Black**: `#0F172A`

### Semantic Colors
- **Success**: `#00864B` (Primary green)
- **Warning**: `#F59E0B` (Professional amber)
- **Error**: `#EF4444` (Professional red)
- **Info**: `#3B82F6` (Professional blue)

## 🏗️ Typography System

- **Headers**: Playfair Display (serif) - Elegant, prestigious
- **Body**: Inter (sans-serif) - Clean, professional, readable

## 📐 Design Principles

✅ **Professional & Trustworthy** - Green conveys education, growth, and stability
✅ **Brand-Aligned** - All colors extracted directly from NGESS logo
✅ **Accessible** - High contrast ratios for readability
✅ **Consistent** - Unified color system across light and dark modes
✅ **Logo-Prominent** - Logo placed on left side of navbar for strong brand presence

## 🔄 Light Mode Theme

- Background: `#FDFDF5` (off-white)
- Glass components: Semi-transparent green overlays
- Text: Dark green (`#0A1F0A`) for primary text
- Accents: Primary green (`#00864B`) with hover effects

## 🌙 Dark Mode Theme

- Background: `#060e07` (forest black)
- Glass components: Semi-transparent white overlays
- Text: White with transparency for hierarchy
- Accents: Primary green (`#00864B`) glowing effects

## 🎯 Component Styling

### Navbar
- Logo: 48x48px, rounded corners, shadow
- Links: Underline indicator on active page
- Admission Button: Solid primary green with white text
- Responsive: Logo text hidden on mobile

### Buttons
- Primary: Solid green background
- Secondary: Transparent with green border
- Hover: Scale transform + opacity change

### Cards
- Glass effect with semi-transparent backgrounds
- Green-tinted borders
- Shadow layers for depth

## 📱 Responsive Design

- Mobile: `375px` - Logo image only, text hidden
- Tablet: `768px` - Full navbar with collapsed menu option
- Desktop: `1024px+` - Full navigation visible
